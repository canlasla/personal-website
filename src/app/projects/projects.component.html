OnFoot

In 2nd year, one of the main projects for ECE297 is to create a Geographic Information System (GIS) similar to Google Maps. My two colleagues and I created OnFoot. This project required extracting data from a database and storing them in appropriate data structure, creating an interactive map, determining optimal travel routes, and solving a variation of the Travelling Saleman Problem. Technologies used to develop this project include C++, GTK Toolkit, and Git for version control. Check out our demo video here.

Reversi AI

This was developed for a school assignment in the Winter Semester of 1st year. I created a minimax algorithm that was optimized with alpha-beta pruning. My AI would play against other students in Reversi to compete for the top spot on a leaderboard. My final placement was 16th out of a pool of 73 students. This was developed in C.

Typing Maniac

This was my first ever computer project! I started programming for a high school course in 2017 where my first language was Python. The end of the semester project was to create anything you want! I decided to create a typing game where words would travel from the top of the screen to the bottom. The player would have to type the word before it would reach the bottom of the screen. If the player was too slow, game over! Unfortunately, the code became deprecated, but I plan to recreate it in the near future.
